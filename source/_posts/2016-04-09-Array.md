# JS判断数组是否有重复项


```sh
function Check(arr)
{
    //将数组转字符串保存至变量b中
    var b = arr.join(",");
    //遍历字符串b
    for (var i = 0; i < b.length; i++) 
    {
        //替换字符串b第一个与arr[i]匹配的字符为""并从头到尾地检索字符arr[i]，有重复项则返回true，否则返回false
        if (b.replace(arr[i],"").indexOf(arr[i])>-1)   
            {
                return true;
            }
    }
    return false;
}   
```
